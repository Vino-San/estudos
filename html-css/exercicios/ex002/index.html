Parágrafos e quebras de linha

Você pode escrever um parágrafos de qualquer jeito. Basta colocar tudo no meio do par de tags <p> e </p>

se precisar quebrar o texto em algum lugar especifico
como esse, voce pode usar a tag <br>

Vamos adicionar símbolos especiais:
®: Marca registrada
©: Copyright
™: Trade Mark
€: Euro
↑: Sera para cima
Δ: Delta maiusculo
δ: denta minusculo

Vamos adicionar emojes:
🦔 Ouriço
👿 Oni
😳 vergonha
😡 Raiva
😻 GATINHA APAIXONADA 😻
